- Add `addupdate` method for asset

```
archetype effect_method_asset_addupdate
...
action exec () {
  effect {
    my_asset.addupdate("my_id", {value = 3})
  }
}
```
is equivalent to
```
archetype effect_method_asset_addupdate
...
action exec () {
  effect {
    if (my_asset.contains("my_id")) then
      my_asset.update("my_id", {value = 3})
    else
      my_asset.add({id = "my_id"; value = 3})
  }
}
```
